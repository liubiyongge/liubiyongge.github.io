# Windows网线共享网络

- 电脑1 : 无线网卡 + 有线网卡
- 电脑2 ：有线网卡
- 网线

1. `电脑1`配置`无线网络`
2. 进入`控制面板\网络和 Internet\网络连接`
3. 用网线连接`电脑1`和`电脑2`
4. 桥接电脑1的`WLAN`和`以太网`
5. 查看`网桥`和`WLAN`使`WLAN`为外网地址，`网桥`为内网地址

> 一般在`网桥->属性->`勾上以太网和WLAN实现

6. 在电脑二中配置网关为网桥的网关，配置IP地址。
