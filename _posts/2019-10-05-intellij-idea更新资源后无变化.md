# intellij idea更新资源后无变化

Intellij Idea:

- Run --> Edit Configurations --> On Update option 和 On Frame Deactivation 都选 Update classes and resources

Chrome

- Chrome右键 --> 检查元素 --> Network --> Disable Cache打钩

- Chrome中 Ctrl+Shift+R 强制刷新页面，不使用缓存
