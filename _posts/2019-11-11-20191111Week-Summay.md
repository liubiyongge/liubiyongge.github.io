## 大学学习方式和上大学以前学习方式



- 背景

进入大学已经两年多，回想自己大学的学习方式感觉和上

大学以前的学习方式差别不大，虽然自己大学基本上都是自学，但是看书的时候还是潜在的以一种应付考试的方式在进行。时刻猜想着考试重点，因为不知道考试重点，所以就算是书上明知道不重要的内容，也当心会考到,强迫自己记下来。今天听到高等代数老师说我们这本书就讲一个内容，发现自己没有把书薄来，只是把书读厚了。有时遇到难点就直接跳过，殊不知这才是你进步的有所获的地方。

- 总结
  - <span style="color:red">抛弃应付考试的思想</span> 要应付考试应该是期末考试前一个到两个星期复习。
  - <span style="color:red">牢记重点</span> 书中一般会给出本书重点，我们要牢记重点，跟着重点走，不要过于纠结在不重要的地方。
  - <span style="color:red">理解才是第一位</span> 一本书很厚，但是真正有收获，真正要花时间的就是几个关键点、难点的理解。不能跳过这些关键的难点。
  - <span style="color:red">把书读厚</span> 深刻了解书中的某些重要细节。
  - <span style="color:red">把书读薄</span> 用一句话概括这本书的内容，总结书中的几个重要的思想。

  tips:

  真正要做笔记的地方是难于理解或容易理解错的地方，这样下次你忘了之后，再次看书的时候能够马上正确理解。

## 查找资料文献

- 背景

这个星期我在做声音的音调检测，首先自己分析，得到的音调一直不符合实际，于是在网上查找相关资料。当然是用英文在google上查找资料，首先我输入pitch detection, 得到了几个相关算法，但是和我面对的问题不一样，于是有用pitch track state of art,搜索到了一个非常有用的针对我的问题的模型。而我们导师查资料就是直接输入几个大牛的名字，看他们的论文。

- 总结
  - <span style="color:red">搜索资料平台</span> 学术文献检索平台 github google youtube(有时会找到一些不错的视频)
  - <span style="color:red">搜索关键词</span> 要把自己的意思用英文表达出来，要斟酌这个单词是不是你想的意思，同时要知道相关术语, 最好先去查一下这个领域的大牛，然后找大牛的文章
  - <span style="color:red">写</span> 每次都要把你遇到的问题写下，可以理清自己思路，在查资料时也会更有章法

## 科研接触新领域

- 背景

每次接触一个新领域的时候，总是查不到自己想要的东西。

- 总结

自己不了解这个领域，所以每次输入搜索词不能得到有用的东西，我们首先应该了解这个领域的发展历史以及现状，以及一些基础知识。
